PROTOGEAR

INDIE / AA GAME CONSOLE

OVERVIEW

ProtoGear is a dedicated platform for independent and AA game development, combining a purpose-built hardware environment with a lightweight, BSD-based operating system. The platform emphasizes predictable performance, developer-friendly tools, and curated game experiences, while remaining affordable and accessible to players.

HARDWARE SPECIFICATIONS

CPU: 6 cores / 12 threads (Zen 2 / Zen 3)
CPU Clock: 3.0–3.3 GHz base, ~4.0 GHz boost
L2 Cache: 3 MB
L3 Cache: 12–16 MB
GPU: RDNA 2 (10–12 CUs)
GPU Clock: 1.6–2.0 GHz
Compute: 2.0–2.5 TFLOPS
Memory: 4 GB DDR4-3200 (dual-channel)
Performance Target: 900p @ 60 FPS
Storage: eMMC for OS, NVMe for games

SOFTWARE PLATFORM

The console runs a BSD-based OS with a custom lightweight window manager. Games execute on the ProtoGear runtime, which mediates access to graphics, audio, input, storage, networking, and memory allocation. Developers use the ProtoGear SDK to target the runtime efficiently.

DEVELOPER FOCUS

ProtoGear emphasizes quality and meaningful content over scope creep. Developers can publish cross-platform or console-exclusive titles, with incentive structures that reward user-generated content, mods, and additional replay value. The console ships ready-to-play, with launch exclusives and a carefully curated initial game library.

PLATFORM OBJECTIVES

  • Deliver a defined, accessible, and stable hardware/software platform.
  • Provide an affordable console package with high-quality games included.
  • Encourage developer creativity while maintaining system constraints.
  • Focus on offline-first gameplay with optional online functionality.
  • Support community content and replay value through mod-friendly systems.
ProtoGear is an independent commercial project under unified creative and technical direction.
